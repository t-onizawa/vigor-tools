# DCAPアジャイルシート v0.1

VIGOR LAB の小ツール v0.1

---

## ツール名

**DCAPアジャイルシート**

---

## 目的

考えすぎて止まっている人を、次の15分のDoに戻すためのシートです。

これは計画表ではありません。行動に戻るためのツールです。  
PDCAではなく、**DoからはじまるDCAP**を使います。

---

## 使い方

1. 「今、止まっていること」を入力する
2. 「本当は進めたいこと」を入力する
3. 「次の15分でできること」を入力する（必須）
4. 「シートを生成する」ボタンを押す
5. 生成されたDoシートを見て、**今すぐ動き始める**

コピーボタンでテキストをクリップボードに保存できます。  
リセットボタンで入力内容をすべて消去します。  
入力内容はどこにも保存されません。

## ローカルで開く方法

`index.html` をブラウザで直接開くだけで動作します。  
サーバー不要、インストール不要です。

```
# macOS の場合
open index.html

# または Finder から index.html をダブルクリック
```

---

## GitHub Pages で公開する方法

1. このディレクトリを GitHub リポジトリとして作成・プッシュする
2. リポジトリの **Settings > Pages** を開く
3. Source を「Deploy from a branch」に設定する
4. Branch を「`main`」、フォルダを「`/ (root)`」に設定して Save する
5. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される
