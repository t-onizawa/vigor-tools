# DCAPアジャイルシート v2

VIGOR LAB の小ツール v2

---

## ツール名

**DCAPアジャイルシート**

---

## 目的

考えすぎて止まっている人を、次の15分のDoに戻すためのシートです。

これは計画表ではありません。行動に戻るためのツールです。  
PDCAではなく、**DoからはじまるDCAP**を使います。

v2 では、DCAp の思想（Do→Check→Act→small plan）をより体現した出力と、  
過去シートを振り返れる履歴機能を追加しています。

---

## v2 の変更点（v0.1 との差分）

- 出力に「次の一手（small plan）」ブロックを追加
- 入力内容に応じた出力パターンの分岐
- localStorage による過去シートの履歴保存（最大10件）
- 履歴はフォーム下に一覧表示

---

## 使い方

1. 「今、止まっていること」を入力する
2. 「本当は進めたいこと」を入力する
3. 「次の15分でできること」を入力する（必須）
4. 「シートを生成する」ボタンを押す
5. 生成されたDoシートを見て、**今すぐ動き始める**

コピーボタンでテキストをクリップボードに保存できます。  
リセットボタンで入力内容をすべて消去します。  
入力内容はどこにも送信されません（端末内のみ）。
